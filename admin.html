TELEGRAM → SITE SYNC
Static HTML can't connect to Telegram directly. To sync a Telegram channel to the site:

Option A — Vercel/Netlify Function
1. Create a Telegram bot via @BotFather, get the API token.
2. Create a serverless function (Node.js) that calls https://api.telegram.org/bot<TOKEN>/getUpdates or sets a webhook.
3. Function parses posts (text, photo, caption) → transforms to {id, date, text, img, tag}.
4. Writes to a public posts.json file or KV store.
5. In index.html, add fetch('/api/posts.json') on load and merge with local posts.

Option B — Notion API
1. Create Notion integration at notion.so/my-integrations.
2. Share your database with the integration.
3. Serverless function calls https://api.notion.com/v1/databases/<DB_ID>/query.
4. Transforms blocks to post format → generates JSON feed.
5. Frontend fetches the feed periodically.

Recommended stack: Vercel (free tier) + Node.js + cron trigger (GitHub Actions) every 30 min.
